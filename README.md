# ngx-material-keyboard
Onscreen virtual keyboard for [Angular] using [Material 2].

## Demo
A demo can be found [here][demo].

## Docs
Generated documentation can be found [here][docs].

## Development
### Repository
The sources of this project are provided by a mono repo containing the library itself and the demo application.
Thus the repositories `demo` and `core` are deprecated and will be removed in the future.

This repository is managed by and layed out for [ng-packagr].

### Versioning
The application uses [semver][SemVer] and is developed with the [git flow branching model][Git-Flow].
